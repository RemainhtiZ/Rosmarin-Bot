# 外矿采集系统（OutMine）功能文档

---

## 概述

外矿系统管理附属房间（非 Controller 房间）的资源采集、建造和防御，包括能量外矿、中央九房、外矿防御等功能。

---

## 核心概念

### 1. 外矿类型

| 类型 | 说明 | 距离 | 防御 |
|------|------|------|------|
| `energy` | 能量外矿 | 1 房 | 单体或双人 |
| `center` | 中央九房 | 0-1 房 | 高威胁 |

### 2. 外矿角色

| 角色 | 说明 | 体型示例 | Boost |
|------|------|----------|-------|
| `out-harvest` | 外矿采集 | w5c1m3 | 无 |
| `out-carry` | 外矿搬运 | c6m6 | 无 |
| `out-mineral` | 矿物采集 | w10m5 | 无 |
| `reserver` | 预定房间 | c1m6 | 无 |
| `out-build` | 外矿建造 | w10m4 | 无 |
| `out-attack` | 中九房攻击 | a10m10 | 无 |
| `out-defend` | 外矿防御 | r4h1m3 | 无 |
| `out-2attack` | 高威胁攻击 | a8t12a20m10 | XUH2O/XGHO2 |
| `out-2heal` | 高威胁治疗 | t12h10m10 | XLHO2/XGHO2 |

### 3. 防御等级

外矿防御根据威胁等级调整：

| 威胁等级 | 说明 | 防御策略 |
|---------|------|----------|
| `LOW` | 少量 NPC | 单体 out-defend |
| `MEDIUM` | 中等 NPC/弱 PC | 单体 out-defend |
| `HIGH` | 大量 NPC/强 PC | 双人小队 out-2attack + out-2heal |
| `CRITICAL` | PowerBank | 不采集，专注清理 |

## 外矿管理

### 1. 外矿扫描

```typescript
const scanOutMines = () => {
    const config = getOutMineConfig();
    const scanned: Set<string> = new Set();

    // 扫描能量外矿
    for (const roomName of config.energy) {
        const room = Game.rooms[roomName];
        if (room && !room.my && room.find(FIND_SOURCES).length > 0) {
            scanned.add(roomName);
        }
    }

    // 扫描中央九房
    for (const roomName of config.centerRoom) {
        const room = Game.rooms[roomName];
        if (room && !room.my && room.controller) {
            scanned.add(roomName);
        }
    }

    return Array.from(scanned);
};
```

### 2. 外矿防御评估

```typescript
const evaluateOutmineDefense = (roomName: string): DefenseLevel => {
    const room = Game.rooms[roomName];
    if (!room) return 'LOW';

    // 检查敌人数量
    const enemies = room.find(FIND_HOSTILE_CREEPS).length;
    const powerEnemies = room.find(FIND_HOSTILE_POWER_CREEPS).length;

    // 检查 Source Keeper
    const sourceKeepers = room.find(FIND_HOSTILE_CREEPS).filter(e =>
        e.owner?.username === 'Source Keeper'
    ).length;

    if (enemies === 0 && powerEnemies === 0) {
        return sourceKeepers > 3 ? 'MEDIUM' : 'LOW';
    }

    if (powerEnemies > 0) return 'CRITICAL';

    if (enemies > 5) return 'HIGH';

    return 'MEDIUM';
};
```

### 3. 外矿任务下发

```typescript
const updateOutMineTasks = () => {
    const outMines = scanOutMines();

    for (const roomName of outMines) {
        const threatLevel = evaluateOutmineDefense(roomName);

        // 高威胁：不采集，专注防御
        if (threatLevel === 'CRITICAL') {
            spawnDefense(roomName, threatLevel);
            continue;
        }

        // 检查是否已控制
        const room = Game.rooms[roomName];
        if (room?.my) {
            // 已控制：正常采集任务
            spawnHarvester(roomName);
            spawnCarrier(roomName);
        } else {
            // 未控制：预定 + 建造
            spawnReserver(roomName);
            spawnOutBuilder(roomName);
        }

        // 防御任务
        spawnDefender(roomName, threatLevel);
    }
};
```

## 角色执行

### 1. out-harvest（外矿采集）

```typescript
const OutHarvest = {
    run: function(creep: Creep) {
        const room = Game.rooms[creep.memory.targetRoom];
        if (!room) {
            creep.moveTo(creep.memory.homeRoom);
            return;
        }

        // 寻找 Source
        const source = room.find(FIND_SOURCES)[0];
        if (!source) {
            // 中央九房：寻找自己房间的 Source
            const homeRoom = Game.rooms[creep.memory.homeRoom];
            if (homeRoom) {
                source = homeRoom.find(FIND_SOURCES)[creep.memory.sourceId];
            }
        }

        if (!source) {
            creep.moveTo(creep.memory.homeRoom);
            return;
        }

        // 采集
        if (creep.pos.isNearTo(source)) {
            creep.harvest(source);
        } else {
            creep.moveTo(source.pos);
        }
    }
};
```

### 2. out-carry（外矿搬运）

```typescript
const OutCarry = {
    run: function(creep: Creep) {
        const room = Game.rooms[creep.memory.targetRoom];
        if (!room) {
            creep.moveTo(creep.memory.homeRoom);
            return;
        }

        // 检查是否需要搬运
        const task = creep.room.getTransportMission(creep);
        if (task) {
            const { sourceObj, targetObj, resourceType } = getTaskData();

            if (creep.store[resourceType] < task.data.amount) {
                if (creep.pos.isNearTo(sourceObj)) {
                    creep.withdraw(sourceObj, resourceType);
                } else {
                    creep.moveTo(sourceObj.pos);
                }
            } else if (creep.store[resourceType] > 0) {
                if (creep.pos.isNearTo(targetObj)) {
                    creep.transfer(targetObj, resourceType);
                } else {
                    creep.moveTo(targetObj.pos);
                }
            }
        } else {
            // 通用搬运：送回 homeRoom
            const homeRoom = Game.rooms[creep.memory.homeRoom];
            const storage = homeRoom?.storage;

            if (storage) {
                for (const res in creep.store) {
                    if (creep.pos.isNearTo(storage)) {
                        creep.transfer(storage, res as ResourceConstant);
                    } else {
                        creep.moveTo(storage.pos);
                    }
                }
            }
        }
    }
};
```

### 3. out-mineral（矿物采集）

```typescript
const OutMineral = {
    run: function(creep: Creep) {
        const room = Game.rooms[creep.memory.targetRoom];
        if (!room) {
            creep.moveTo(creep.memory.homeRoom);
            return;
        }

        // 寻找 Mineral
        const mineral = room.find(FIND_MINERALS)[0];
        if (!mineral) {
            creep.moveTo(creep.memory.homeRoom);
            return;
        }

        // 检查是否有 extractor
        const extractor = room.find(FIND_MY_STRUCTURES, {
            filter: s => s.structureType === STRUCTURE_EXTRACTOR
        });

        if (!extractor) {
            // 等待 extractor 建造
            creep.moveTo(mineral.pos);
            return;
        }

        // 采集
        if (creep.pos.isNearTo(mineral)) {
            const result = creep.harvest(mineral);
            if (result !== OK && result !== ERR_TIRED) {
                // 采集失败，可能需要移动
                creep.moveTo(mineral.pos);
            }
        } else {
            creep.moveTo(mineral.pos);
        }
    }
};
```

### 4. reserver（预定房间）

```typescript
const Reserver = {
    run: function(creep: Creep) {
        const room = Game.rooms[creep.memory.targetRoom];
        if (!room) {
            creep.moveTo(creep.memory.targetRoom);
            return;
        }

        const controller = room.controller;
        if (!controller) {
            creep.moveTo(creep.memory.homeRoom);
            return;
        }

        // 检查是否需要预定
        const ticksToDowngrade = controller.ticksToDowngrade || 0;
        if (ticksToDowngrade < CONTROLLER_DOWNGRADE[controller.level]) {
            // 立即预定
            if (creep.pos.inRangeTo(controller, 1)) {
                creep.reserveController(controller);
            } else {
                creep.moveTo(controller.pos);
            }
            return;
        }

        // 检查预定等级
        const reservation = room.controller?.reservation;
        if (!reservation || reservation.ticksToEnd < 2000) {
            // 需要延长预定
            if (creep.pos.inRangeTo(controller, 1)) {
                creep.reserveController(controller);
            } else {
                creep.moveTo(controller.pos);
            }
            return;
        }

        // 预定充足，回家休息
        creep.moveTo(creep.memory.homeRoom);
    }
};
```

### 5. out-defend（外矿防御）

```typescript
const OutDefend = {
    run: function(creep: Creep) {
        const room = Game.rooms[creep.memory.targetRoom];
        if (!room) {
            creep.moveTo(creep.memory.targetRoom);
            return;
        }

        // 寻找敌人
        const enemies = room.find(FIND_HOSTILE_CREEPS);
        const powerEnemies = room.find(FIND_HOSTILE_POWER_CREEPS);

        // 优先攻击 PowerCreep
        let target = powerEnemies[0];
        if (!target && enemies.length > 0) {
            // 没有敌人，清理 NPC
            const invader = room.find(FIND_HOSTILE_STRUCTURES)[0];
            if (invader) {
                target = invader as any;
            }
        }

        if (target) {
            // 攻击
            if (creep.pos.inRangeTo(target, 3)) {
                creep.rangedAttack(target);
                if (getEnemyCount(creep, 3) >= 3) {
                    creep.rangedMassAttack();
                }
            } else {
                creep.moveTo(target.pos);
            }
        } else {
            // 巡逻
            patrol(creep);
        }
    }
};
```

### 1. 道路建造

```typescript
const updateExternalRoads = () => {
    const outMines = scanOutMines();

    for (const roomName of outMines) {
        const room = Game.rooms[roomName];
        if (!room?.my) continue;

        // 检查是否有外矿建造任务
        const builders = room.find(FIND_MY_CREEPS, {
            filter: c => c.memory.role === 'out-build'

        if (builders.length === 0) continue;

        // 建造从 homeRoom 到 source 的路
        const homeRoom = Game.rooms[creep.memory.homeRoom];

};
```

## 优缺点分析

### 优点

1. **自动化**：自动扫描和管理外矿房间
2. **分级防御**：根据威胁等级动态调整防御强度
3. **资源优先**：优先采集高价值资源
4. **路网支持**：自动建造和维护外矿道路
5. **安全保护**：Source Keeper 和 PowerCreep 自动清理

### 缺点

1. **资源消耗**：外矿采集消耗大量能量和时间
2. **Creep 损失**：高威胁外矿 Creep 损失率高
3. **路网维护**：外矿道路需要持续维护
4. **依赖预定**：未控制房间依赖预定机制
5. **Boost 依赖**：高威胁场景依赖 Boost 资源

## 注意事项

1. **Source Keeper**：注意 Source Keeper 的攻击范围（3 格）
2. **PowerBank**：PowerBank 出现时暂停采集，专注清理
3. **预定冲突**：避免与其他玩家预定同一房间
4. **路网规划**：合理规划道路，减少建造和维护成本
5. **能量消耗**：外矿操作消耗能量，确保家房间有足够能量
6. **Creep 转移**：外矿 Creep 死亡后及时补充
